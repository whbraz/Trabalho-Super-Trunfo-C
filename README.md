# Super Trunfo em C

Trabalho Super Trunfo em C: Desenvolvendo a Lógica do Jogo

## Descrição

Este programa foi desenvolvido em linguagem C como parte de um trabalho acadêmico.
O objetivo é simular o jogo Super Trunfo, permitindo o cadastro de duas cartas e a comparação de atributos entre elas.

O usuário informa os dados de cada carta e escolhe dois atributos diferentes para comparação.
A carta vencedora é definida pela soma dos valores dos atributos escolhidos.

## Atributos disponíveis para comparação

- População
- Área
- PIB
- Pontos Turísticos
- Densidade Demográfica

## Como compilar

Utilize um compilador C, como o GCC.

Exemplo no terminal:

gcc super_trunfo.c -o super_trunfo

## Como executar

Após compilar, execute o programa com o comando:

./super_trunfo

## Exemplo de uso

1. O programa solicita os dados da Carta 1.
2. Em seguida, solicita os dados da Carta 2.
3. O usuário escolhe o primeiro atributo para comparação.
4. O usuário escolhe o segundo atributo para comparação.
5. O programa exibe os valores, a soma dos atributos e a carta vencedora.

## Observações

- Os atributos escolhidos para comparação não podem ser iguais.
- O programa é executado via terminal.
